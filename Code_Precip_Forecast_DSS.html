PrecForecDSS:


Acquiring precipitation forecast and its conversion to DSS

Description

PrecForecDSS acquires precipitation forecast and converts acquired data to DSS format. The acquired precipitation is the bias-corrected Global Forecast System (GFS) for a given lead time. The acquired data is automatically projected to the Cypress Creek Watershed, Texas. The precipitation map is automatically generated in the folder "Forecast_GFS" with the name precip_plot.pdf. The DSS file is automatically generated in the folder "Forecast_GFS" with the name GFS.dss. The script Forecast_GFS.py re-samples the precipitation to a 1000 m Ã— 1000 m grid cell and 1 hour time step.



Snapshots of some PrecForecDSS results

Capabilities

  • Acquires precipitation (bias-corrected Global Forecast System) for a given lead time

  • Acquired data is automatically projected to watershed of interest

  • Precip data is converted to DSS format for its use in HEC-HMS
